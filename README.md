# Todo App (PHP + MySQL)

This repository contains a simple procedural PHP to-do application with a user dashboard and administrative backend.

## Quick Start

1. Import `install/install.sql` into your MySQL server (database `todo_app`).
2. Update the connection credentials inside `config/db.php` if necessary.
3. Run the built-in PHP server: `php -S localhost:8000 -t .`
4. Visit `http://localhost:8000/install/seed.php` once to insert demo accounts and sample tasks.
5. Open `http://localhost:8000` in your browser to begin using the app.

### Default Accounts

- Admin: `admin / admin`
- User: `jane / password`

Enjoy managing your tasks!
